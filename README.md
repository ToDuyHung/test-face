# Face Recognition — ArcFace R50 Test Suite

Pipeline: **MTCNN** (detect + 5-landmark align) → **ArcFace R50** (embed) → **cosine similarity**

---

## Project Structure

```
test-face/
├── test3/
│   ├── gallery/          # Source images (67 persons × 5 poses = 335 images)
│   └── probes/           # Probe images for ad-hoc testing
│
├── benchmark/            # Auto-generated by prepare_benchmark.py
│   ├── gallery/          # Enrolled persons – front pose only (47 persons)
│   ├── known/            # Other poses of enrolled persons (188 images, should match)
│   └── strange/          # Un-enrolled persons (100 images, should be rejected)
│
├── models/
│   ├── w600k_r50.onnx         # ArcFace R50 ONNX (auto-downloaded on first run)
│   └── arcface_r50.mlpackage  # CoreML version (generated by convert_to_coreml.py)
│
├── test_pytorch.py        # Ad-hoc probe test using ArcFace ONNX

│
├── prepare_benchmark.py   # Split test3/gallery → benchmark/{gallery,known,strange}
├── benchmark.py           # Benchmark: ArcFace R50 ONNX       (Linux/Windows/Mac)
├── benchmark_resnet.py    # Benchmark: InceptionResnetV1/VGGFace2 (Linux/Windows/Mac)

├── convert_to_coreml.py   # Convert w600k_r50.onnx → arcface_r50.mlpackage
├── infer_coreml.py        # Single-image or gallery/probe inference via CoreML
│
├── README.md
└── requirements.txt
```

## CoreML Conversion (macOS)

Convert the ONNX model to `.mlpackage` for iOS/macOS deployment:

```bash
pip install onnx2torch onnx
python convert_to_coreml.py --precision float16
```

Run inference with the CoreML model:

```bash
# Single image
python infer_coreml.py --image test3/probes/FID70025400_1.png

# Gallery vs probes
python infer_coreml.py --gallery benchmark/gallery --probes benchmark/known
python infer_coreml.py --gallery benchmark/gallery --probes benchmark/strange --threshold 0.5
```

## Benchmark Metrics Explained

| Metric | Description |
|---|---|
| **FAR** | False Acceptance Rate — strangers accepted / total strangers |
| **FRR** | False Rejection Rate — known persons rejected / total known |
| **TAR** | True Acceptance Rate = 1 − FRR |
| **ACC(known)** | Among accepted known probes, % that matched the correct identity |
| **EER** | Equal Error Rate point (FAR ≈ FRR) — balanced operating point |
| **FAR=0** | Smallest threshold where zero strangers are accepted |
| **AUC** | Area Under ROC curve (1.0 = perfect separation) |

### Recommended thresholds

| Model | Threshold |
|---|---|
| ArcFace R50 ONNX | ~0.5 |
| ArcFace R50 CoreML | same as ONNX |

## Model Details

### ArcFace R50 (`w600k_r50.onnx`)
- Source: InsightFace `buffalo_l` — [GitHub releases](https://github.com/deepinsight/insightface/releases/tag/v0.7)
- Input: RGB 112×112, normalised to `[-1, 1]`
- Output: 512-d L2-normalised embedding
- Preprocessing: `pixel_norm = (pixel / 127.5) − 1.0`

### InceptionResnetV1 (`facenet_pytorch`)
- Source: `facenet_pytorch` pre-trained on VGGFace2
- Input: RGB 160×160 (MTCNN aligned, `post_process=True`)
- Output: 512-d L2-normalised embedding

### Face alignment
Both models use **MTCNN** for face detection + 5-point landmark extraction,  
followed by **umeyama affine warp** to the canonical face crop.
